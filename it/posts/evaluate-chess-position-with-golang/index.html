Valutare le posizioni scacchistiche è una parte importante della strategia scacchistica e può aiutare i giocatori a prendere decisioni più informate sulle mosse da fare. In questo articolo vedremo come usare il potente motore scacchistico Stockfish e il linguaggio di programmazione Go per valutare posizioni di scacchi.

Installare Stockfish
#

Prima di poter usare Stockfish, è necessario installarlo. Su Linux e macOS, si può installare Stockfish con il proprio package manager (ad esempio, apt-get, Homebrew, ecc.). Su Windows, si può scaricare un eseguibile precompilato dal sito di Stockfish (https://stockfishchess.org/download/).

Usare Stockfish con Go
#

Per usare Stockfish con Go occorre una libreria che fornisca un’interfaccia Go verso il motore. Tra le opzioni disponibili, in questo esempio useremo la libreria “github.com/notnil/chess”.

Per installarla, eseguire il seguente comando:

go get github.com/notnil/chess

Siamo ora pronti a usare Stockfish nei nostri programmi Go.

Valutare una posizione scacchistica
#

Per ottenere la valutazione di Stockfish su una posizione, per prima cosa occorre creare una nuova istanza di Stockfish:

stockfish, err := chess.NewEngine("stockfish")
if err != nil {
    panic(err)
}

Poi si può impostare il livello di difficoltà, che controlla la profondità di ricerca di Stockfish per la mossa migliore. Un livello più alto produrrà valutazioni più precise, ma richiederà più tempo di calcolo.

stockfish.SetDifficulty(20)

Ora si può impostare la posizione da valutare usando la notazione scacchistica standard (ad esempio, “rnbqkbnr/pppppppp/8/8/8/8/PPPPPPPP/RNBQKBNR w KQkq - 0 1” per la posizione iniziale).

stockfish.SetPosition("rnbqkbnr/pppppppp/8/8/8/8/PPPPPPPP/RNBQKBNR w KQkq - 0 1")

Infine, si chiama il metodo Evaluate() per ottenere la valutazione di Stockfish sulla posizione:

evaluation, err := stockfish.Evaluate()
if err != nil {
    panic(err)
}

fmt.Println(evaluation)  // Outputs: "0.00"

La valutazione viene restituita come stringa in centipedine, dove un valore positivo indica che il Bianco è in vantaggio e un valore negativo indica che il Nero è in vantaggio. Un valore di “0.00” indica una posizione equilibrata.

Codice completo
#

import "github.com/notnil/chess"

func main() {
    // Create a new Stockfish instance
    stockfish, err := chess.NewEngine("stockfish")
    if err != nil {
        panic(err)
    }

    // Set the difficulty level (this controls how deep Stockfish will search)
    stockfish.SetDifficulty(20)

    // Set the position to be evaluated
    stockfish.SetPosition("rnbqkbnr/pppppppp/8/8/8/8/PPPPPPPP/RNBQKBNR w KQkq - 0 1")

    // Get Stockfish's evaluation of the position
    evaluation, err := stockfish.Evaluate()
    if err != nil {
        panic(err)
    }

    fmt.Println(evaluation)  // Outputs: "0.00"
}

Questo è solo un esempio di base; con la libreria si possono fare molte altre cose (eseguire mosse, ottenere la lista delle mosse legali, ecc.). Ulteriori informazioni sulla libreria e le sue funzionalità sono disponibili nella documentazione all’indirizzo https://godoc.org/github.com/notnil/chess.

Conclusione
#

In questo articolo abbiamo visto come usare il motore scacchistico Stockfish e il linguaggio di programmazione Go per valutare posizioni di scacchi. Si tratta di uno strumento utile sia per i giocatori che vogliono migliorare il proprio gioco, sia per i developer che vogliono costruire applicazioni legate agli scacchi. Con poche righe di codice Go, è possibile ottenere valutazioni potenti di qualsiasi posizione scacchistica da Stockfish.
